|---|---|---|
| U2 | `=IFERROR(VLOOKUP(D2,$T$3:$U$4,2,FALSE),"")` | 検索条件選択番号変換 |
| X3 | `=IF(SlectNo=1,V3,IF(SlectNo=2,W3,""))` | 選択番号に応じた検索条件1表示 |
| X4 | `=IF(SlectNo=1,V4,IF(SlectNo=2,W4,""))` | 選択番号に応じた検索条件2表示 |
| B4 | `=X3` | 検索開始条件 |
| D4 | `=X4` | 検索終了条件 |
| K4 | `=OvTani` | 線量単位参照 |
| N5 | `=SenNoE-SenNoS` | 照射時間 |

| AH11〜AH行 | `=P*AG` | 補正後オーブン線量（管理点との比 × 補正管理点線量） |
| AI11 | `=AH11` | 累積積算線量（1行目） |
| AI12〜AI行 | `=AI(前行)+AH` | 累積積算線量（累積加算） |

> **温度補正計算式の解説**:
> `(-3.2478 + 23.386 × 測定値(ABS)/素子厚さ(mm)) × (-0.006 × 測定温度(℃) + 1.0558)`
>
> - 第1因子: フィルム線量計の線量変換式（ABS値/厚さから管理点線量を算出）
> - 第2因子: 温度補正係数（標準温度からの補正、線形近似）
>
> これは固体飛跡検出器（フィルム型線量計）の温度依存性補正計算式であり、照射中のオーブン加熱温度による線量計感度変化を補正する。

## 6. VBA モジュール仕様

### 6.1 実績.bas

#### `実績検索()`

**処理概要**: 受付番号（`Uno`）で在庫/在庫履歴テーブルを検索し、会社情報と受付番号一覧を実績シートに表示する。

### 6.5 Sheet5.cls（管理点線量シート）

#### `温度補正()`

**処理概要**: 管理点線量シートの測定データ行に対して温度補正計算式を直接セルに書き込む。

```vba
Sub 温度補正()
    Dim row As Integer
    row = 11
    Do Until Cells(row, 8) = ""
        ' 管理点線量 = (-3.2478 + 23.386 × ABS値/厚さ) × (-0.006 × 温度 + 1.0558)
        Cells(row, 9).Value = (-3.2478 + 23.386 * Cells(row, 8) / Cells(row, 7)) * (-0.006 * Cells(row, 11) + 1.0558)
        Cells(row, 17).Value = Cells(row, 9) * Cells(row, 16)
        If row = 11 Then
            Cells(row, 17).Value = Cells(row, 17)
        Else
            Cells(row, 17).Value = Cells(row, 17) + Cells(row - 1, 17)
        End If
        row = row + 1
    Loop
End Sub
```

#### `線量率()`

**処理概要**: Q列（製品積算線量）の最終値 ÷ N5（照射時間）× 1000 で線量率を計算し O5 セルに表示する。

### 6.6 変換10進を2進.bas

#### `ExHenkan(Number) As String`

**処理概要**: 10進数を2進数文字列に変換する。

```vba
Function ExHenkan(ByVal Number) As String
    Dim myNumber As Single
    myNumber = Number
    Do Until myNumber < 1
        ExHenkan = myNumber Mod 2 & ExHenkan
        myNumber = Int(myNumber / 2)
    Loop
End Function
```
