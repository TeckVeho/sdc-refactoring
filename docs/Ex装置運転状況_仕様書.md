# Ex装置運転状況 仕様書

> **ファイル種別**: .xlsm（マクロ付き）
> **用途**: 照射装置（1号機・2号機・3号機）の稼働時間・運転実績を照射管理DBから集計・グラフ表示し、１号機のイベント履歴を詳細表示するツール
> **VBA プロジェクトサイズ**: 約377KB
> **外部連携**: DSN=ricdb（Oracle DB）、グラフ（chart1.xml）

Private Sub Workbook_Open()

End Sub

**処理概要**: 保存ダイアログを抑止して閉じる。

### 6.2 SQL_Execution

**概要**: ADODB を使った DB 接続・SQL 実行の共通モジュール。

#### `Open_oraconDB()`

**処理概要**: `DSN=ricdb;UID=ric;PWD=t6101` でDB接続を開く。

```vba
Sub Open_oraconDB()
    oraconn.ConnectionString = "DSN=ricdb;UID=ric;PWD=t6101"
    oraconn.Open
    oraconn.CursorLocation = adUseClient
End Sub
```

#### `SQL_Exe(mySQL)`

**処理概要**: SQL 文を実行し結果を `rs` レコードセットに格納する。

#### `SQL_INSERT_UPDATE(myTBL, myKey, myD(), myN)`

**処理概要**: 指定テーブルを COUNT で確認し、0件なら INSERT、1件以上なら UPDATE を実行してトランザクションをコミットする。

#### `SQL_Delete(myTBL, myWhere)`

**処理概要**: 指定テーブルの条件に一致するレコードを DELETE してコミットする。

#### `Disp_Sheet(mySQL, mySH, myRow, myRecordCount, myColumn, myFieldCount, myF)`

**処理概要**: SQLを実行し、結果をワークシート `mySH` の指定行列に `CopyFromRecordset` で貼り付ける。

#### `Set_Array(mySQL, myData(), myRecordCount, myFldCount)`

**処理概要**: SQLを実行し、結果を2次元配列 `myData(i,j)` に格納する（i=レコード番号, j=フィールド番号）。

### 6.3 運転時間

#### `装置稼働時間表示()`

**処理概要**: 入力値を検証し、1〜3号機の稼働時間を集計して稼働時間シートに表示する。1日指定時はグラフ・前日/翌日ボタンを表示する。

**処理フロー**:
1. 集計開始日・終了日の入力値検証（空白・日付形式・未来日チェック、6か月以内チェック）
2. 総時間（`TotalTime`）を算出
3. 1日指定の場合はグラフと前日/翌日ボタンを可視化
4. `Ric1Kadou` / `Ric2Kadou` / `Ric3Kadou` を呼び出し各号機の稼働時間を計算

#### `前日表示()` / `翌日表示()`

**処理概要**: `KaisiDay` を±1日して `装置稼働時間表示` を再実行する。

#### `Ric1Kadou()` （Private）

**処理概要**: DB テーブル `sengnr1` から1号機のイベントデータを取得し、照射時間・停止時間を算出して稼働時間シートに設定する。グラフ表示時は `R1Graph` でトレンドデータを生成する。

**主要 SQL**:
```sql
-- 開始時直前のイベント
SELECT MAX(sdate||stime) FROM sengnr1
WHERE sdate||stime < '<開始日>000000'

-- 期間内イベント
SELECT sdate||' '||stime, TO_NUMBER(timer), event FROM sengnr1
WHERE sdate||stime >= '<前回最終>' AND sdate||stime <= '<終了日>240000'
ORDER BY sdate||stime

-- 移動照射時間の合計
SELECT SUM(TO_NUMBER(timer)) FROM sengnr1
WHERE TO_NUMBER(timer) < 24
  AND sdate||stime >= '<開始日>' AND sdate||stime < '<終了日>'
```

#### `Ric2Kadou()` （Private）

**処理概要**: DB テーブル `kyouj2` から2号機のデータを取得し、停止時間を算出する。

**主要 SQL**:
```sql
-- 期間内データ有無
SELECT COUNT(*) FROM kyouj2
WHERE rectime >= '<開始日> 000000' AND rectime < '<終了日> 240000'

-- 期間内の実速度データ
SELECT RECTIME, REALSPD FROM KYOUJ2
WHERE rectime >= '<前回最終>' AND rectime <= '<終了日> 240000'
ORDER BY rectime
```

#### `Ric3Kadou()` （Private）

**処理概要**: DB テーブル `sengnr3` から3号機のデータを取得し、照射時間・停止時間を算出する。イベントはビット演算（`ExHenkan`）でデコードする。

**主要 SQL**:
```sql
-- 集計開始時の最終レコード
SELECT MAX(sdate) FROM sengnr3
WHERE sdate < TO_DATE('<開始日>','yyyy/mm/dd hh24:mi:ss')

-- 期間内データ
SELECT sdate, sekitime, event FROM sengnr3
WHERE sdate >= TO_DATE('<最小日>','yyyy/mm/dd hh24:mi:ss')
  AND sdate <= TO_DATE('<最大日>','yyyy/mm/dd hh24:mi:ss')
ORDER BY sdate
```

#### `R1Graph(myD(), myRecNo)` / `R2Graph(myD(), myRecNo)` / `R3Graph(myD(), myRecNo)` （Private）

**処理概要**: 各号機の1分刻みトレンドデータを `G_Data` の対応列（D/E/F）に書き込む。1号機は 0/0.5/1、2号機は 1.2/2.2、3号機は 2.4/3.4 でグラフ値を表現する。

#### `ExHenkan(Number)` （Private Function）

**処理概要**: 10進数を2進数文字列に変換する（3号機イベントのビットマスク解析用）。

#### `ChengTime(DTime)` （Private Function）

**処理概要**: `"YYYYMMDDHHMMSS"` 形式の文字列を Excel シリアル日時値に変換する。

### 6.4 イベント集計

#### `イベント別集計()`

**処理概要**: 指定期間の1号機イベント別発生回数を `sengnr1` から集計し、1号機集計表シートの `RIC1Kekka` 領域に表示する。

**処理フロー**:
1. `Yori`・`Made` の入力チェック
2. 日付を `YYYYMMDD` 形式に変換
3. DBから `event` 別の件数を取得
4. `RIC1Kekka` に書き込み

```sql
SELECT event, COUNT(*) FROM sengnr1
WHERE sdate >= '<開始日>' AND sdate <= '<終了日>'
GROUP BY event
ORDER BY event
```

#### `詳細表示()`

**処理概要**: 指定期間の1号機イベント詳細（日付/時刻/タイマー/イベント）を取得し、H列以降に表示してイベント内容・照射時間を付加する。10,000件超の場合はエラーメッセージを表示して中断する。

```sql
-- 件数チェック
SELECT COUNT(*) FROM sengnr1
WHERE sdate >= '<開始日>' AND sdate <= '<終了日>'
ORDER BY sdate, stime

-- 詳細データ取得
SELECT sdate 日付, stime 時刻, timer タイマー, event イベント
FROM sengnr1
WHERE sdate >= '<開始日>' AND sdate <= '<終了日>'
```

#### `Hours(myDay, myHor)` （Private Function）

**処理概要**: 日付文字列（`YYYYMMDD`）と時刻文字列（`HHMMSS`）をExcelシリアル日時値に変換する。

### 6.5 クリア

#### `スタート()`

**処理概要**: イベントを一時無効にして `初期画面` を呼び出す。`Workbook_Open` から呼ばれる。

#### `初期画面()` （Private）

**処理概要**: 1号機集計表・稼働時間シートの入力値・表示データをクリアし、シートをシートを保護状態（`UserInterfaceOnly:=True`）にする。グラフ・前日/翌日ボタンを非表示にする。

#### `クリア()`

**処理概要**: `Kekka`・`Yori`・`Made` の各名前付き範囲をクリアする。

## 8. データフロー

```
【稼働時間集計フロー】
  稼働時間シート: 集計開始日・終了日 入力
        ↓ [実行]ボタン → 装置稼働時間表示()
  入力値検証（空白・日付形式・未来日・6か月以内）
        ↓
  総時間(TotalTime) = (終了日+1 − 開始日) × 24
        ↓
  Ric1Kadou()
    DB(sengnr1): イベント履歴取得 → タイマー計算 → SyousyaTime[1号機]
    1日指定時 → R1Graph() → G_Data のD列にトレンドデータ書込
        ↓
  Ric2Kadou()
    DB(kyouj2): 稼働状態取得 → 停止時間計算 → SyousyaTime[2号機]
    1日指定時 → R2Graph() → G_Data のE列にトレンドデータ書込
        ↓
  Ric3Kadou()
    DB(sengnr3): イベント取得 → ビット解析 → SyousyaTime[3号機]
    1日指定時 → R3Graph() → G_Data のF列にトレンドデータ書込
        ↓
  稼働時間シート D8:F8 の数式で稼働率を自動計算
  （1日指定時: chart1 グラフが G_Data をもとに可視化）

【1号機イベント集計フロー】
  1号機集計表: 開始日・終了日 入力
        ↓ [回数集計]ボタン → イベント別集計()
  DB(sengnr1): GROUP BY event → RIC1Kekka に書込
        ↓ [詳細表示]ボタン → 詳細表示()
  DB(sengnr1): 詳細データ取得 → H列以降に表示
  イベントコードを EvetList でデコード → L列(イベント内容), M列(照射時間)
```

## 9. セキュリティ注意事項

| 種別 | キーワード | 内容 |
|---|---|---|
| AutoExec | `Workbook_Open` | ブックを開くと `スタート()` が自動実行され初期画面が設定される |
| AutoExec | `Workbook_BeforeClose` | ブックを閉じる際に保存ダイアログを抑止する |
| AutoExec | `Worksheet_Change` | 稼働時間シートのセル変更時に自動的にイベントが発火する |
| Suspicious | `Open` | ファイル操作処理が含まれる（DB接続の Open が該当） |
| Suspicious | `Call` | モジュール間の内部呼び出し（DLL呼び出しではない） |
