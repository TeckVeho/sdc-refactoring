# Ex１号機作業指図書 仕様書

> **ファイル種別**: .xlsm（マクロ付き）
> **用途**: 1号機の照射作業指図書を作成するツール。受付番号をスキャンして在庫データをDBから取得し、照射条件（線量・寸法・台高さ等）を入力して照射時間を計算し、作業指図書を印刷する。線量率データの登録・参照機能も内蔵。
> **VBA プロジェクトサイズ**: 12モジュール（ThisWorkbook.cls / Sheet1〜4.cls / 開始処理実行.bas / 箱長さ記録.bas / 線量率処理.bas / 共通変数.bas / 関数Function.bas / 画面クリア_保護.bas / 印刷出力.bas / データ取得書込み.bas / チェック計算実行.bas / セット図表示非表示.bas）

│   ├── 照射時間計算
│   ├── 画面初期設定
│   └── 計算結果出力（DBOUT）
└── ボタン（線量率登録シート 3個）
    ├── 登録
    ├── 表示
    └── 線量率登録終了

### 6.2 Sheet1.cls（指図書シート）

#### `Worksheet_BeforeRightClick(Target, Cancel)`

**処理概要**: 右クリックで特定セルを「□」⇔「■」でトグル切替する。対象セルは照射方法（I38〜I41）・径（P42・T42）・反転方法（K45・O45・S45）・線量計（I47・O47・S47・O50・S50・I51）・ラベル（I53・P53・R53・U53）・温度管理（I55・Q55）・顧客指示（I57・L57）。

**セット方法トグルの詳細ロジック**:
- I38（1号機セット方法）が■になったとき: 他の3つ□に・矢印コネクタ10表示
- I39（シート算出）が■になったとき: 厚さ>2cmの場合は確認ダイアログ・矢印コネクタ106表示
- I40（その他）が■になったとき: 全矢印コネクタ非表示
- I41（線量率測定）が■になったとき: 矢印コネクタ108表示

#### `Worksheet_Change(Target)`

**処理概要**: セル変更時の連動処理。

| 変更セル | 処理内容 |
|---|---|
| E3（作成日） | 開始日が空なら開始日に作成日をセット、期間終了日更新、線量率読込 |

### 6.9 データ取得書込み.bas

#### `受付データ取得()`

**処理概要**: 受付番号（Uno）から在庫データをDBから取得し、会社名・指定線量・上下限線量・入荷数・入荷日をシートに表示する。バーコードスキャン（16文字）とキー入力（確認ダイアログあり）を区別。

**SQL**:
```sql
SELECT z.kaisyacd, z.kainame, TO_NUMBER(z.siteisn),
       TO_NUMBER(z.jyougsn), TO_NUMBER(z.kagensn), z.labelcd,
       z.labelht, TO_NUMBER(z.misyousu), TO_NUMBER(z.nyukasu),
       z.nyukabi
FROM zaiko z
WHERE z.syouso = '1' AND z.uno = '<受付番号>'
```

#### `線量率ファクター読込()`

**処理概要**: 照射開始日から月末日までの `ratetbl` データを設定値シートの `RateTBL` 範囲に取得する。変更有無を確認して `RateData`（G4:G9）に線量率をセットする。

**SQL**:
```sql
SELECT KIKANS, rate60, rate50, rate40, rate30, rate20, dm1
FROM ratetbl
WHERE kikans >= '<KaisiBi>'
  AND kikans <= '<Eday>'
```

#### `線量率データ登録()`

**処理概要**: 線量率登録シートの入力値を `ratetbl` に開始日〜終了日の日別で INSERT/UPDATE する。

**SQL**:
```sql
-- 日別に繰り返し実行
INSERT INTO ratetbl (kikans, rate20, rate30, rate40, rate50, rate60, dm1, tbq, syainid, wdate, kikane)
VALUES ('<YYYY/MM/DD>', '<rate20>', '<rate30>', '<rate40>', '<rate50>', '<rate60>', '<dm1>', <tbq>, '<id>', '<now>', '          ')
-- または UPDATE ratetbl SET rate20=..., ... WHERE kikans='<YYYY/MM/DD>'
```

#### `線量率データ読込()`

**処理概要**: 指定期間の `ratetbl` データを DB から取得し、変更点ごとにグループ化して線量率登録シートに表示する（最大3変更グループまで）。

#### `Syain_read()`

**処理概要**: 社員番号を InputBox で入力させ、`shainmst` から社員名を取得して「作成者」セルにセットする。

**SQL**:
```sql
SELECT TRIM(REPLACE(shaname, '　', ''))
FROM shainmst
WHERE shano = '<社員番号>'
```

### 6.10 チェック計算実行.bas

#### `照射時間計算実行()`

**処理概要**: DB線量率登録確認 → 入力チェック → セット方法またはシート照射に応じた照射時間計算を実行する。

#### `チェック(mpErrDes)`

**処理概要**: 全入力項目の妥当性を検証する。エラー時は `mpErrDes` にメッセージをセットして終了。チェック項目:
照射開始期間・DB線量率データ整合・作成者・受付番号・重量・保護板距離・箱寸法・セット方法・線量率測定線量計・反転回数・反転方法・線量計有無・線量計位置・指定線量・上限線量・下限線量・管理ラベル・CI有無・温度管理・顧客先指示

#### `時間計算(mpErrDes)`

**処理概要**: 1号機セット方法またはシート照射以外の場合の照射時間計算。

- セット方法（SetHouhou=■）: 台高さ → 線量率 → 補正係数（高さ<50cm かつ 重量<5kgなら0.9、それ以外1.0）→ 照射時間 = RoundUp(SiteiSn / mpDoseR, 3)
- 線量率測定（SetRitu=■）: 測定線量率（SenRitu）→ 照射時間 = RoundUp(SiteiSn / SenRitu, 4)

#### `シート照射時間(mpErrDes)`

**処理概要**: シート計算（SheetCal=■）の場合の照射時間計算。台高さ20cm固定・箱高さ40cm以下のみ有効。

計算式:
- 線量率 = 10 / SheetHr （kGy/h）
- 照射時間 = RoundUp(SiteiSn / SenRituS, 4)

## 8. データフロー

```
【起動フロー】
  ブックを開く（Workbook_Open）
       ↓ 開始処理()
  指示書保護（3シートをUIのみ保護）
  画面消去初期化（入力欄クリア・初期値設定）
  Syain_read → InputBox で社員番号入力
       ↓ DB（shainmst）: 社員名取得
  作成日・開始日 = 本日
  線量率ファクター読込
       ↓ DB（ratetbl）: 開始日〜月末の線量率取得 → 設定値シート RateTBL に格納

【受付番号入力フロー】
  E4 に受付番号入力（バーコードスキャン / キー入力）
       ↓ Worksheet_Change → 受付データ取得()
  DB（zaiko）: syouso='1' の在庫データ取得
       ↓
  会社名・指定線量・上下限線量・入荷数・入荷日をシートに表示

【照射時間計算フロー】
  箱寸法（幅・長さ・高さ）・台高さ・重量を入力
       ↓ 台高さ入力で台高さ() 関数が mpDoseR を設定
  「照射時間計算」ボタン押下
       ↓ チェック() で全入力項目バリデーション
  セット方法=■ → 時間計算()
    台高さ → 線量率 → 補正係数 → SenRituS / SetJikanS に結果表示
  シート算出=■ → シート照射時間()
    10kGy照射時間 / 線量率 → SetJikanS に結果表示
  線量率測定=■ → 時間計算() （SenRitu入力から）
    SenRitu → SetJikan に結果表示

【印刷フロー】
  「指図書印刷」ボタン押下
       ↓ Workbook_BeforePrint → 長さ記録()
  DB（ExKeikakuX）: 箱長さを UPDATE/INSERT
       ↓ チェック() バリデーション
  印刷ダイアログ表示

【線量率登録フロー】
  「線量率登録」ボタン → 線量率登録シートへ移動
  RaDateS / RaDateE に期間入力、各Rate20〜Rate60 + シート + 線源量を入力
  「登録」ボタン → 線量率データ登録()
       ↓ DB（ratetbl）: 日別で INSERT/UPDATE
  「線量率登録終了」ボタン → 指図書シートに戻る
```

## 9. セキュリティ注意事項

| 種別 | キーワード | 内容 |
|---|---|---|
| AutoExec | `Workbook_Open` | 起動時に自動で社員番号入力ダイアログが表示され DB 接続が行われる |
| AutoExec | `Workbook_BeforeClose` | 閉じる前に保存フラグが自動操作される |
| AutoExec | `Workbook_BeforePrint` | 印刷前に自動でDB書き込み処理が実行される |
| AutoExec | `Worksheet_Change` | セル変更時に自動で DB 照会・計算処理が実行される |
| Suspicious | `Call` | Excel 4 マクロ形式の DLL 呼び出しの可能性 |
| Suspicious | `Hex/Base64 Strings` | エンコード文字列が存在 |

> **注意**: DSN=ricdb の接続文字列（UID: ric / PWD: t6101）が `データ取得書込み.bas` の `Syain_read` 内にハードコードされています。
> **注意**: `HakoMax`・`OmosaMax` の名前付き範囲が `#REF!` になっています。設定値シートの参照先が削除されている可能性があります。
