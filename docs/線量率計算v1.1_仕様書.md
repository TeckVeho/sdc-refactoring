### 2.2 積算INTEG

**目的**: 積算型電位計（nC 読み値）を使用した積算線量（Gy）の計算。線量率RATEと同構造で単位・係数が異なる。

#### 主要相違点（線量率RATEとの比較）

| 項目 | 線量率RATE | 積算INTEG |
|---|---|---|
| 電位計読み値 | pA（電流） | nC（電荷積算） |
| 最終変換係数 | ×3.6（pA→nC/h変換） | なし（nCそのまま） |
| 計算結果 | 線量率 [Gy/h] | 積算線量 [Gy] |
| 結果セル | H37 | G42:H43 |
| 電位計校正定数 | S6: 1.0013, S7: 1.0 | R6: 0.9996, R7: 0.9997 |
| 不確かさ参照 | T31から計算 | S31から計算（INTEG用不確かさ） |

#### レイアウト構造（主要部分）

| セル範囲 | 内容 |
|---|---|
| B1 | タイトル「積算INTEG ver.1.1」 |
| G7:H8 | 電位計読み値入力（nC） |
| G42:H43 | **積算線量計算結果**（Gy、小数点以下3桁目で切り捨て） |
| H46 | 確認用計算（切り捨てなし） |
| P6-P20 | 電離箱選択番号（1〜9） |
| T6 | 電位計選択番号入力 |
| T11 | 電離箱選択番号入力 |

## 4. 数式一覧

### 線量率RATEシート

| セル | 数式 | 説明 |
|---|---|---|
| V6 | `=CHOOSE(U6,S6,S7)` | U6の選択番号で電位計校正定数を選択 |
| E6 | `=V6` | 使用係数（電位計）の表示 |
| V11 | `=CHOOSE(U11,S12,S13,S14,S15,S16,S17,S18,S19,S20)` | U11の選択番号で電離箱校正定数を選択（9択） |
| E12 | `=V11` | 使用係数（電離箱）の表示 |
| X11 | `=X15/Y15` | Cs空気→Co水換算係数 = Co水(0.02965)/Cs空気(0.02932) = 1.0113 |
| S17 | `=ROUNDDOWN(T17*X11,6)` | #1200のCo水換算後校正定数（6桁切捨て） |
| S18 | `=ROUNDDOWN(T18*X11,7)` | #1202のCo水換算後校正定数（7桁切捨て） |
| S19 | `=ROUND(T19*X11,7)` | #1203のCo水換算後校正定数（7桁四捨五入） |
| S20 | `=ROUNDDOWN(T20*X11,8)` | #1205のCo水換算後校正定数（8桁切捨て） |
| S24 | `=ROUNDDOWN((273.2+H24)/(273.2+22)*(1013.3/H26),4)` | 温度気圧補正係数 κTP（標準T0=22℃、P0=1013.3hPa） |
| V24 | `=CHOOSE(U24,S24,S25)` | 温度気圧補正係数の選択（1=あり: κTP計算値、2=なし: 1） |
| E25 | `=IF(V24=1,"なし",V24)` | 使用係数（温度気圧補正）の表示 |
| T31 | `=SQRT(Y32^2+AD31^2)` | 合成不確かさ% = √(電位計不確かさ²+電離箱不確かさ²) |
| S31 | `=ROUNDDOWN(1-(T31/100),4)` | 合成不確かさ係数（ANTM機関選択時、4桁切捨て） |
| T32 | `=SQRT(Y32^2+AD36^2)` | 合成不確かさ%（JQA機関選択時） |
| S32 | `=ROUNDDOWN(1-(T32/100),4)` | 合成不確かさ係数（JQA機関選択時） |
| V31 | `=CHOOSE(U31,S31,S32,S33)` | 不確かさ係数の選択（1=ANTM, 2=JQA, 3=なし） |
| E31 | `=IF(V31=1,"なし",V31)` | 使用係数（不確かさ）の表示 |
| V37 | `=CHOOSE(U37,S37,S38)` | ケーブル補正係数の選択（1=あり: 0.8991、2=なし: 1） |
| E37 | `=IF(V37=1,"なし",V37)` | 使用係数（ケーブル補正）の表示 |
| K40 | `=Co空気/Co水` | ケーブル補正係数計算式参照 (= 0.02666/0.02965 = 0.8991) |
| **H37** | `=ROUNDDOWN(V6*V11*V24*G7*V31*V37*3.6,(3)-INT(LOG(V6*V11*V24*G7*V31*V37*3.6)))` | **線量率 [Gy/h]**。小数点以下3桁目になるよう桁数を動的決定して切り捨て |
| H43 | `=ROUNDUP(C43/H37,1)` | **照射時間 [h]** = 設定線量 / 線量率（小数点以下第2位切り上げ） |
| H46 | `=V6*V11*V24*V31*G7*V37*3.6` | 確認用計算（切り捨てなし） |

#### 線量率計算式の展開

```
線量率 [Gy/h] =
  電位計校正定数 (V6)
  × 電離箱校正定数 (V11) [Gy/nC]
  × 温度気圧補正係数 κTP (V24)
  × 電位計読み値 (G7) [pA]
  × 不確かさ係数 (V31)
  × ケーブル補正係数 (V37)
  × 3.6  ← 1pA = 3.6nC/h 換算（1A=3600C/h）
```

| セル | 数式 | 説明 |
|---|---|---|
| U6 | `=CHOOSE(T6,R6,R7)` | 電位計校正定数選択 |
| E6 | `=U6` | 使用係数（電位計）の表示 |
| U11 | `=CHOOSE(T11,R12,R13,R14,R15,R16,R17,R18,R19,R20)` | 電離箱校正定数選択（9択） |
| W11 | `=W15/X15` | Cs空気→Co水換算係数 |
| R17 | `=ROUNDDOWN(S17*W11,6)` | Co水換算後校正定数 |
| R24 | `=ROUNDDOWN((273.2+H24)/(273.2+22)*(1013.3/H26),4)` | 温度気圧補正係数 κTP |
| U24 | `=CHOOSE(T24,R24,R25)` | 補正係数選択 |
| S31 | `=SQRT(X32^2+AC31^2)` | 合成不確かさ%（ANTM） |
| R31 | `=ROUNDDOWN(1-(S31/100),4)` | 合成不確かさ係数（ANTM） |
| U31 | `=CHOOSE(T31,R31,R32,R33)` | 不確かさ係数選択 |
| U37 | `=CHOOSE(T37,R37,R38)` | ケーブル補正選択 |

| **G42** | `=ROUNDDOWN(U6*U11*U24*G7*U31*U37,(3)-INT(LOG(U6*U11*U24*G7*U31*U37)))` | **積算線量 [Gy]**（×3.6なし） |
| H46 | `=U6*U11*U24*U31*G7*U37` | 確認用計算 |

#### 積算線量計算式の展開

```
積算線量 [Gy] =
  電位計校正定数 (U6)
  × 電離箱校正定数 (U11) [Gy/nC]
  × 温度気圧補正係数 κTP (U24)
  × 電位計読み値 (G7) [nC]
  × 不確かさ係数 (U31)
  × ケーブル補正係数 (U37)
  ← ×3.6 不要（nC 読み値なので単位変換不要）
```

## 8. データフロー

```
【線量率計算フロー（線量率RATEシート）】

1. 電位計選択
   U6 に選択番号入力
       ↓ CHOOSE(U6, S6, S7)
   V6 = 電位計校正定数（例: 1.0013）

2. 電離箱選択
   U11 に選択番号入力
       ↓ CHOOSE(U11, S12...S20)
   V11 = 電離箱校正定数 [Gy/nC]
   （Cs空気校正機器は S17〜S20: ROUNDDOWN(T*X11,...) で Co水換算済）

3. 電位計読み値入力
   G7 に読み値（pA）を入力

4. 温度気圧補正（任意）
   H24=温度(℃)、H26=気圧(hPa) を入力
       ↓ S24 = ROUNDDOWN((273.2+T)/(273.2+22)*(1013.3/P), 4)
   U24 で補正あり/なし選択 → V24 = κTP

5. 不確かさ選択
   U31 で機関選択（ANTM/JQA/なし）
       ↓ T31 = SQRT(電位計不確かさ²+電離箱不確かさ²)
       ↓ S31 = ROUNDDOWN(1-(T31/100), 4)
   V31 = 合成不確かさ係数

6. ケーブル補正選択
   U37 で あり/なし → V37 = 係数（0.8991 or 1）

7. 最終計算
   H37 = ROUNDDOWN(V6×V11×V24×G7×V31×V37×3.6, 動的桁)
        → 線量率 [Gy/h]

8. 照射時間（オプション）
   C43 に設定線量（Gy）入力
   H43 = ROUNDUP(C43/H37, 1)
        → 照射時間 [h]

【積算線量計算フロー（積算INTEGシート）】
  上記と同構造。
  G7 に読み値（nC）を入力し、×3.6 なしで計算。
  G42 = ROUNDDOWN(U6×U11×U24×G7×U31×U37, 動的桁)
        → 積算線量 [Gy]
```
